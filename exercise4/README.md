# AI Content Studio
This Node.js project generates a complete content suite from a topic using AI. It creates **articles, summaries, social media posts, images, and audio narrations**, then saves everything in `content-suite.json`.

---

## Features

- Generate **articles** on any topic.  
- Create **short summaries** in 3 sentences.  
- Write **social media posts** (Facebook-style).  
- Generate **header and thumbnail images**.  
- Produce **audio narration** for summaries.  
- Automatically handles **rate limiting** from APIs.  

---

## AI Models Used

- **Text Generation**: `meta/meta-llama-3-8b-instruct`  
- **Image Generation**: `google/imagen-4-fast`  
- **Audio Generation**: `minimax/speech-02-turbo`
